# 工具箱页面使用说明

## 概述
工具箱页面集成了常用的在线开发工具，方便开发者快速进行各种格式转换和数据处理操作。

## 功能特性

### 1. Properties ↔ YAML 互转工具
- **功能**: 在 Properties 和 YAML 格式之间互相转换
- **使用**: 
  - 选择转换方向（Properties → YAML 或 YAML → Properties）
  - 在左侧输入框中输入源格式内容
  - 点击"转换"按钮，结果将显示在右侧输出框
  - 支持复制输出结果到剪贴板

### 2. JSON 格式化工具
- **功能**: JSON 字符串的格式化和压缩
- **使用**:
  - 在输入框中输入 JSON 字符串
  - 点击"格式化"按钮进行美化格式化
  - 点击"压缩"按钮进行压缩（去除多余空格）
  - 支持复制格式化结果

### 3. Base64 编码/解码工具
- **功能**: 字符串的 Base64 编码和解码
- **使用**:
  - 在输入框中输入需要处理的字符串
  - 点击"编码"按钮进行 Base64 编码
  - 点击"解码"按钮进行 Base64 解码
  - 支持复制处理结果

### 4. URL 编码/解码工具
- **功能**: URL 字符串的编码和解码
- **使用**:
  - 在输入框中输入需要处理的 URL 或字符串
  - 点击"编码"按钮进行 URL 编码
  - 点击"解码"按钮进行 URL 解码
  - 支持复制处理结果

### 5. 时间戳转换工具
- **功能**: 时间戳与日期时间的互相转换
- **使用**:
  - 左侧显示当前时间的秒级和毫秒级时间戳
  - 右侧输入时间戳，点击"转换"按钮获取对应的日期时间
  - 支持复制时间戳和转换结果

### 6. UUID 生成器
- **功能**: 生成 UUID v1 和 v4 格式的唯一标识符
- **使用**:
  - 选择 UUID 版本（v1 或 v4）
  - 设置生成数量（1-10个）
  - 点击"生成 UUID"按钮生成唯一标识符
  - 支持复制生成的 UUID

## 技术实现

### 路由配置
```javascript
{
  path: 'tool_box',
  name: 'tool_box',
  component: () => import('@/views/tool_box/Index.vue'),
  meta: {
    label: '工具箱',
    breadcrumbName: ['工具箱', '在线工具']
  }
}
```

### 页面结构
- **顶部操作栏**: 工具类型选择下拉框
- **工具内容区域**: 根据选择的工具显示对应的操作界面
- **响应式设计**: 适配不同屏幕尺寸

### 主要组件
- 使用 Ant Design Vue 组件库
- 支持复制到剪贴板功能
- 错误处理和用户提示
- 实时时间戳更新

## 使用说明

1. **访问工具箱页面**: 通过系统菜单导航到工具箱页面
2. **选择工具类型**: 从下拉列表中选择需要的工具
3. **输入数据**: 在左侧输入框中输入需要处理的数据
4. **执行操作**: 点击相应的操作按钮（转换、格式化、编码等）
5. **获取结果**: 在右侧输出框查看处理结果
6. **复制结果**: 点击"复制"按钮将结果复制到剪贴板

## 注意事项

- 输入数据时请确保格式正确
- 大文件处理可能会影响性能
- 某些转换操作可能有格式限制
- 建议在使用前备份重要数据

## 扩展性

工具箱页面设计为可扩展架构，可以轻松添加新的工具类型：

1. 在工具选择下拉框中添加新选项
2. 在模板中添加对应的工具面板
3. 在脚本中添加相应的处理逻辑
4. 添加必要的样式和布局

这种模块化设计使得新工具的集成变得简单高效。
